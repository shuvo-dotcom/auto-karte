How to Use

Voice Commands

  • • "Select nuclear" - Show all nuclear power plants
  • • "Select 4 solar" - Select 4 solar facilities
  • • "Connect all" - Connect selected facilities with lines
  • • "Clear" - Clear all selections
  • • "Filter gas" - Show only gas power plants
Voice Issues? Try these solutions:
  • • Chrome/Edge: Works best with these browsers
  • • HTTPS: Voice works better on HTTPS sites
  • • Microphone: Allow microphone access when prompted
  • • Network: Ensure stable internet connection
  • • Fallback: Use text commands if voice fails

Text Commands

  • • Type commands in the input field
  • • Use quick select buttons for instant selection
  • • Click on map markers to select/deselect
  • • Use "Connect All" button to link facilities
